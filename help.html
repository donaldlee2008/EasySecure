Encryption
  • Go to The Encryption Window By Clicking on the Icon.
  • In the Window Select the file or the Folder to be Encrypted
  • Choose either of the AES Algorithm (128 bit key) or the Blow fish ALgorithm
  • Select the password with which you want to encrypt your file
  • You can either generate a key file for additional security or keep using the password for simplicity
  • If you chose only the password, then you have nothing to worry. Your file is encrypted with one of the most secure algorithms in the world. Your work is done!
  • If you chose for a key file to be generated, select the key file during decryption.Your file is encrypted with one of the most secure algorithms in the world along with your password and key file
Decryption
  • Select either the encrypted file to be decrypted
  • If you have encrypted your file with only a password, just enter your password and decrypt your file back
  • If you have encrypted your file along with a key file, just enter your key file and decrypt your file back
Shred a File(Use with Caution)
  • If you want your file and all the copies of your file in a folder to be completely destroyed beyond recognition so that wont be recovered even by a forensic tool go to this option
  • Select the file to be shreded and the folder from where all the copies of the file are to be shreded and rest assured you wont see your file again
Verify Originality
  • If you want to check if your file has been tampered or not, Select this icon. Even if a single character is misplaced, we will let you know, dont worry
  • Select the original file and another copy of the file against which its originality has to be validated
  • If both of the files are exactly same you will be notified by a 'tick' or else by a 'cross'
Upload
  • If you want to upload your file and keep a backup of it, select this option
  • For a first time user, sign in with your email id and password and select the file to be uploaded
  • For an already existing user, login in with your email id and password and select the file to be uploaded
Download
  • For an already existing user, login in with your email id and password and select the file to be downloaded
